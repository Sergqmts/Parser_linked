# Сборщик Ссылок на Python

## Описание
Этот проект представляет собой простой скрипт на Python, который извлекает ссылки из заданного URL с помощью библиотеки BeautifulSoup. Скрипт отправляет GET-запрос, обрабатывает полученный HTML и сохраняет все ссылки, соответствующие определенному тегу и параметру, в текстовом файле.

## Требования
Перед запуском скрипта убедитесь, что у вас установлен Python. Также необходимо установить необходимые библиотеки:

- `requests`
- `beautifulsoup4`

Вы можете установить их с помощью pip:
        pip install requests beautifulsoup4
        pip install requests


## Установка
1. Скачайте или клонируйте этот репозиторий на свой компьютер.
2. Убедитесь, что у вас установлен Python и все необходимые зависимости.

## Использование
1. Откройте файл `extraction.py` (или другой, в котором находится код).
2. В функции `main()` укажите URL-адрес сайта, с которого хотите извлечь ссылки. По умолчанию используется `https://primgazeta.ru/news/`.
3. Укажите тег, который вы хотите использовать для поиска ссылок. По умолчанию используется тег `a`.
        url = 'https://primgazeta.ru/news/'  # Укажите свой URL
        tag = 'a'  # Укажите тег, по которому хотите искать
4. Запустите скрипт.
5. Все извлеченные ссылки будут сохранены в файл `links.txt` в той же директории, где находится ваш скрипт.
6. Подготовьте файл `links.txt`, содержащий список ссылок, которые вы хотите проверить. Каждая ссылка должна быть на отдельной строке.
7. Запустите скрипт: filter.py
8. Все рабочие ссылки будут сохранены в файл `working_links.txt` в той же директории, где находится ваш скрипт.



## Примечания
- Убедитесь, что указанный вами URL доступен и существует.
- Параметр в `link['href']` можно уточнить, добавив условие в список. Например, если вы хотите искать ссылки, содержащие определённое слово.
- Скрипт проверяет статус код каждой ссылки и считает ссылку рабочей, если статус код находится в диапазоне 200-399.
- В случае ошибок при выполнении запросов (например, из-за недоступности ресурса) скрипт будет выводить сообщение о том, что ссылка не работает.
